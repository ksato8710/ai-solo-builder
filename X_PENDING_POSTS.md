## 使用方法

### 手動投稿実行
1. 上記の「未投稿」セクションから投稿内容をコピー
2. X(Twitter) にアクセス
3. 投稿内容を貼り付けて投稿
4. 完了したら該当項目を「完了待ち」に移動

### スクリプト実行（Browser Relay利用時）
```bash
cd /Users/satokeita/dev/ai-navigator
python3 scripts/execute_pending_posts.py
```
