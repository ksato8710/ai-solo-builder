# Claude Code

Anthropicが開発したコーディング特化のAIアシスタント。ブラウザ上でターミナル操作、コード編集、実行環境を統合して提供し、開発プロセス全体を支援します。

## 主要機能

- **統合開発環境**: ブラウザ内でのコーディング・実行
- **ターミナル操作**: Linux/Mac環境のコマンド実行
- **ファイル管理**: プロジェクトファイルの作成・編集
- **実行環境**: Python・Node.js・各種言語の実行

- プロトタイプの高速開発
