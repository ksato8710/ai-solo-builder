## 概要

GitHub CopilotがAnthropicのClaude AgentとOpenAIのCodexを正式にサードパーティエージェントとして統合しました。2026年3月1日より、Enterprise/Pro+に先行提供されていた機能がBusiness/Proユーザーにも開放されています。

**出典:** [GitHub Release Notes](https://releasebot.io/updates/github) — 2026-03-01

## 詳細

### マルチエージェント体制の確立

VS Code 1.109では、以下の3つのエージェントを同一のサブスクリプション内で並列実行できるようになりました。

- **Copilot:** GitHubネイティブのエージェント
- **Claude Agent:** Anthropicの推論重視モデル
- **Codex:** OpenAIのコード特化モデル

### プラン別の利用状況

| プラン | 月額 | Claude/Codex利用 |
|--------|------|------------------|
| Free | $0 | 対象外 |
| Pro | $10 | ✅ 利用可能 |
| Business | $19/user | ✅ 利用可能 |
| Enterprise | $39/user | ✅ 先行提供済み |

### Copilotユーザー数の推移

GitHub Copilotの有料契約者数は**470万人**に達し、前年比75%増加しています。

## ソロビルダーへの示唆

**$10で3つのエージェントを使い分け**

Proプラン（月額$10）で、Copilot・Claude・Codexの3つのエージェントが利用可能になったことは、ソロビルダーにとって大きなコストメリットです。

- **コード補完:** Copilot（速度重視）
- **複雑な推論:** Claude（精度重視）
- **リファクタリング:** Codex（コード特化）

タスクに応じて使い分けることで、コストを抑えながら最適な結果を得られます。既存のVS Code環境をそのまま活用できる点も、移行コストゼロで始められる利点です。
