## TL;DR

**Figma Fan** は、Figmaの新機能リリースをリアルタイムで追跡するコミュニティサイト。2026年2月の新機能6件（MCP連携、Figma Make、エンタープライズ機能など）を**タイムライン形式**で一覧でき、各機能ごとにビデオ、記事、チュートリアル、Reddit議論へのリンクがまとまっている。

---

## Figma Fanとは

Figmaのエコシステムが急速に拡大する中、新機能の把握が難しくなっている。**Figma Fan**は、この課題を解決するコミュニティ主導のリソースサイトだ。

### 主な特徴

| 項目 | 内容 |
|------|------|
| **機能数** | 6件（2026年2月時点） |
| **リソース数** | 27件 |
| **カテゴリ** | 4つ（AI/MCP、Figma Make、Collaboration、Enterprise） |

---

## 最新機能タイムライン（2026年2月）

### 1. Codex to Figma（2/26）
**カテゴリ:** AI/MCP

CodexからMCP経由でFigmaデザインを直接生成。コードロジックから視覚デザインへのシームレスな移行を実現。

**コミュニティリソース:**
- Figma MCP Server — AI Meets Design Tools（動画）
- The Verge: Figma introduces its own MCP server
- Figma Help Center: Getting Started with the Figma MCP Server

---

### 2. FigJam to Notion（2/24）
**カテゴリ:** Collaboration

FigJamの付箋やダイアグラムをMCP連携でNotionデータベースに自動同期。ブレストからドキュメント化までをシームレスに。

**コミュニティリソース:**
- Keep Productive: FigJam + Notion: The Ultimate Workflow
- Figma Blog: FigJam紹介記事

---

### 3. Workspace Sharing（2/23）
**カテゴリ:** Enterprise

ワークスペース全体とファイルをワンクリックで共有。アクセス管理を効率化。

**コミュニティリソース:**
- Figma公式: How to Manage Teams and Permissions
- Spotify Design: Scaling Figma for Large Design Teams

---

### 4. AI Hosting Controls（2/23）
**カテゴリ:** Enterprise

AIトラフィックルーティングの新しいエンタープライズコントロール。管理者がデータフローをより細かく監視可能に。

**コミュニティリソース:**
- TechCrunch: Figma adds AI design tools
- Figma Help Center: Admin controls for AI features

---

### 5. Make Connectors（2/20）
**カテゴリ:** Figma Make

Figma Makeを**Amplitude、Dovetail、zeroheight**に接続。プロダクト開発ワークフローを効率化。

**コミュニティリソース:**
- Figma公式: Figma Make — Build Web Apps From Designs
- UX Collective: Figma enters the code generation space with Make
- DesignCourse: Figma Make First Look

---

### 6. Claude Code to Figma（2/17）
**カテゴリ:** AI/MCP

Claude Codeを使用してWebページをキャプチャし、編集可能なFigmaフレームに即座に変換。

**コミュニティリソース:**
- Fireship: Claude + Figma MCP: Code to Design Instantly（142いいね）
- Smashing Magazine: How AI is changing the design-to-code workflow
- Nielsen Norman Group: The state of AI + design tools in 2026

## ソロ開発者にとっての価値

### 1. 情報収集の効率化

Figmaの新機能は公式ブログ、YouTube、Reddit、各種メディアに分散している。Figma Fanはこれらを**一箇所に集約**し、機能ごとにグルーピングしている。

### 2. 学習リソースへの直接アクセス

各機能に対して：
- 📹 **動画チュートリアル**（YouTube）
- 📰 **解説記事**（メディア）
- 📚 **公式ドキュメント**（Help Center）
- 💬 **コミュニティ議論**（Reddit）

がまとまっているため、学習コストが大幅に削減される。

### 3. MCP連携の最新動向

特にソロ開発者にとって重要な**AI/MCP連携機能**（Codex to Figma、Claude Code to Figma）の情報が充実。AIコーディングツールとデザインツールの統合が加速していることがわかる。

## カテゴリ別の注目ポイント

| カテゴリ | ソロ開発者への影響 |
|----------|-------------------|
| **AI/MCP** | Claude Code/Codexからデザイン生成が可能に。コード→デザインの双方向ワークフロー |
| **Figma Make** | デザインからWebアプリ生成。v0との比較議論も活発 |
| **Collaboration** | FigJam→Notion連携でブレスト→ドキュメント化を自動化 |
| **Enterprise** | AI機能のガバナンス強化。大規模チーム向けだが、セキュリティ意識の参考に |

## 使い方

1. **トップページ**でタイムライン形式の新機能一覧を確認
2. **カテゴリフィルター**（ALL/AI/MK/CO/EN/DS）で絞り込み
3. 各機能の「**Community**」セクションで関連リソースにアクセス
4. 「**Official Announcement**」リンクで公式発表を確認

## まとめ

Figma Fanは、Figmaエコシステムの急速な進化を追いかけるための**必須ブックマーク**になりそうだ。

特に、AIコーディングツール（Claude Code、Codex）とFigmaのMCP連携が進む中、**コード↔デザインの双方向ワークフロー**がどう進化しているかを把握するのに最適。

ソロ開発者として、週1回チェックするだけでFigma関連の最新動向をキャッチアップできる。
